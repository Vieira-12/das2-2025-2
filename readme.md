Aula 08/10/2025
RBAC: controle de acesso simples baseado em papel/cargo do usuário.​

ABAC: controle de acesso mais detalhado baseado em vários atributos e contexto.​

Identidade federada / SSO: usar uma única conta confiável para entrar em vários sistemas.

Criptografia simétrica: mesma chave para cifrar e decifrar, rápida para grandes volumes de dados.​

Criptografia assimétrica: par de chaves pública/privada, ideal para troca segura pela internet (ex.: HTTPS).​

SAST: análise automática do código-fonte para achar vulnerabilidades antes da execução.​

IDS: monitora tráfego em tempo real para detectar atividades suspeitas e gerar alertas.​

VPC Peering: ligação privada entre duas VPCs para comunicação interna sem usar a internet pública.​

Métricas: números que mostram a “saúde” do sistema (uso de CPU, latência etc.).​

Logs: registros detalhados de eventos e erros que aconteceram no sistema.​

Elasticidade: aumentar/diminuir recursos automaticamente conforme a demanda.​

Escalabilidade horizontal: crescer adicionando mais máquinas em paralelo (scale out).​

Balanceador de carga: distribui requisições entre vários servidores e desvia de servidores com falha.​
